## 🧭 总体架构

## 📅 阶段一：基础理论与 API 实践（第 1 个月）

### 第 1 周：Transformer 架构与 OpenClaw 入门

#### 周一：环境准备
- **Notion**: 创建"大模型学习"数据库
- **NotebookLM**: 新建笔记本"LLM Study"，上传论文摘要
- **OpenClaw**: 检查模型列表
- **验收**: Notion 有第一条记录，NotebookLM 可检索论文，OpenClaw 正常

#### 周二：Transformer 原理
- **理论学习**: 阅读 Attention 论文摘要 + 图解 (20min)
- **知识沉淀**: 生成"Transformer 核心概念"卡片 (20min)
- **验收**: 能用自己的话解释 Attention 机制

#### 周三：OpenClaw 基础
- **动手实践**: 调用 Gemini 提问 (25min)
- **动手实践**: 用 Qwen 写 Python 函数 (20min)
- **验收**: 成功调用模型，函数代码正确

#### 周四：提示工程入门
- **理论学习**: 学习零样本/少样本/思维链 (20min)
- **动手实践**: 创建 PromptExpert 智能体 (25min)
- **验收**: 能写出 3 种提示模板

#### 周五：实践 - 智能客服
- **动手实践**: 创建客服智能体 (30min)
- **动手实践**: 飞书群测试 (20min)
- **验收**: 客服智能体能正确回答至少 5 个预设问题

#### 周末：复盘
- **术语表**: 整理≥10 个术语
- **周总结**: NotebookLM 生成
- **验收**: 术语表完成，总结清晰

---

### 第 2 周：模型对比与 API 管理

#### 周一：模型对比
- **动手实践**: 调用 3 个模型问同一问题
- **知识沉淀**: 记录速度/质量/成本差异
- **验收**: 形成对比表格

#### 周二：API 配额管理
- **理论学习**: 学习用量监控命令
- **动手实践**: 配置默认模型+fallback
- **验收**: fallback 生效

#### 周三：本地模型 Ollama
- **动手实践**: 运行 qwen2.5:7b
- **动手实践**: 对比本地 vs 云端
- **验收**: 本地模型正常回复

#### 周四：代码生成实践
- **动手实践**: 生成 TikTok 爬虫脚本
- **动手实践**: 运行脚本抓取数据
- **验收**: 输出≥5 个商品标题

#### 周五：智能体路由
- **动手实践**: 创建 fast-assistant 和 power-assistant
- **动手实践**: 配置飞书群绑定
- **验收**: 飞书群能正常切换智能体

#### 周末：复盘
- **API 最佳实践**: 整理清单
- **模型选型指南**: NotebookLM 生成
- **验收**: 指南包含≥3 个场景推荐

### 第 3 周：RAG 理论入门

#### 周一：RAG 概念
- **理论学习**: 阅读 RAG 相关文章
- **知识沉淀**: 生成 RAG 摘要
- **验收**: 能说出 3 个核心组件

#### 周二：向量数据库
- **理论学习**: 学习 Chroma 使用
- **动手实践**: 创建向量库并检索
- **验收**: 检索返回相似度最高文本

#### 周三：嵌入模型
- **理论学习**: 学习嵌入模型原理
- **动手实践**: 调用嵌入 API
- **验收**: 记录两种模型差异

#### 周四：简单 RAG 实现
- **动手实践**: 搭建最小 RAG 系统
- **动手实践**: 测试 TK 运营文档
- **验收**: 能回答"选品标准是什么"

#### 周五：集成到 OpenClaw
- **动手实践**: 封装 RAG 为技能
- **动手实践**: 飞书群测试
- **验收**: 群内输入"/rag 选品标准"得到回答

#### 周末：复盘
- **RAG 优化点**: 记录≥3 个
- **RAG 实验记录**: NotebookLM 创建
- **验收**: 优化点已记录

---

### 第 4 周：微调基础

#### 周一：微调原理
- **理论学习**: 学习 LoRA/QLoRA 原理
- **知识沉淀**: 微调 vs 提示工程对比
- **验收**: 能解释 LoRA 为何高效

#### 周二：数据准备
- **动手实践**: 收集 20 条典型对话
- **动手实践**: 制作 JSON 数据集
- **验收**: 数据集格式正确

#### 周三：环境搭建
- **理论学习**: 学习 AutoDL/Colab 使用
- **动手实践**: 租用 GPU，安装 Unsloth
- **验收**: 能加载预训练模型

#### 周四：第一次微调
- **动手实践**: LoRA 微调（10 步）
- **动手实践**: 保存 adapter
- **验收**: 微调无报错，生成 adapter

#### 周五：测试微调效果
- **动手实践**: 加载微调模型
- **动手实践**: 对比微调前后回答
- **验收**: 微调后回答更贴近业务

#### 周末：复盘
- **微调实验步骤**: 整理
- **微调最佳实践**: NotebookLM 创建
- **验收**: 形成可复用清单
