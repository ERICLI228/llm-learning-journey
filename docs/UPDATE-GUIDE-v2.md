## 📋 Notion 更新步骤（10 分钟）

### Step 1: 添加新字段（5 分钟）

打开 Notion 数据库，点击右侧 **"+"** 添加以下 7 个新字段：

| # | 字段名 | 类型 | 说明 |
|---|--------|------|------|
| 1 | 任务类型 | Select | 理论学习 / 动手实践 / 项目 / 复盘 |
| 2 | 完成度 | Number | 0-100% |
| 3 | 关联笔记 | Relation | 链接到学习笔记页面 |
| 4 | 实验记录 | Relation | 链接到实验记录页面 |
| 5 | NotebookLM 链接 | URL | NotebookLM 对话链接 |
| 6 | 反思总结 | Text | 完成后反思 |
| 7 | 明日计划 | Text | 下一步行动 |

**添加步骤**：
1. 打开数据库
2. 点击最右侧的 **"+"**
3. 选择字段类型（Select/Number/URL/Text）
4. 输入字段名
5. 保存

**创建步骤**：
1. 在 Notion 左侧边栏点击 **"+ Add a page"**

### Step 2: 使用高效提问模板

保存以下提问模板，学习时直接使用：

#### 概念理解
```
请用简单的语言解释 [概念]
请用比喻解释 [概念]
[概念 A] 和 [概念 B] 有什么区别？
这个概念在 [实际场景] 中如何应用？
```

#### 代码理解
```
这段代码的作用是什么？[粘贴代码]
有没有更优的实现方式？
这段代码的关键步骤是什么？
```

#### 对比分析
```
对比 [方案 A] 和 [方案 B] 的优缺点
[模型 A] 和 [模型 B] 在 [任务] 上各有什么优劣？
基于我的实验数据，分析两者的差异
```

#### 错误排查
```
我遇到了以下错误：[错误信息]
相关代码是：[代码]
可能的原因有哪些？
如何解决？
```

#### 学习材料生成
```
根据我上传的所有笔记，生成一份知识点总结
生成一份 [主题] 的面试题库
生成一份 [主题] 的复习大纲
```

---

### Step 3: NotebookLM + Notion 联动

```
1. 在 Notion 中创建学习任务
   ↓
2. 在 NotebookLM 中上传相关资料
   ↓
3. 学习并向 NotebookLM 提问
   ↓
4. 复制 NotebookLM 的优质回答到 Notion 笔记
   ↓
5. 在 Notion 中更新任务状态
```
